Menu-Driven Calculator with Exception Handling

Overview

This project is a simple menu-driven calculator developed in Python. It allows users to perform basic arithmetic operations through a menu interface while using exception handling to prevent the program from crashing due to invalid input.

Features

- Menu-driven interface
- Addition
- Subtraction
- Multiplication
- Division
- Handles invalid menu choices
- Prevents division by zero
- Handles invalid user input using exception handling
- Allows multiple operations until the user chooses to exit

Technologies Used

- Python 3

How to Run

1. Save the program as "menu_calculator.py".
2. Open a terminal or command prompt.
3. Navigate to the folder containing the file.
4. Run the following command:
   python menu_calculator.py
5. Select an option from the menu and follow the prompts.

Example

===== MENU =====
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Exit

Enter your choice: 1
Enter first number: 15
Enter second number: 10
Result = 25.0

Exception Handling

- "ValueError" – Handles invalid numeric input.
- "ZeroDivisionError" – Prevents division by zero.
- "Exception" – Catches any unexpected errors.
